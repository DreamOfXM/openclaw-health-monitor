Desktop Prototype

OpenClaw Health Monitor

这个桌面壳会优先检查本机 Dashboard 是否已经启动。检测到本地服务后，会自动跳转到监控中心。

首次启动

  1. 先在本机准备好 OpenClaw Health Monitor 仓库。
  2. 运行 `make start` 或 `./start.sh` 启动本地 Guardian 和 Dashboard。
  3. 桌面 App 会自动探测 `127.0.0.1:8080-8089`，找到后自动进入监控中心。

常用命令

cd ~/openclaw-health-monitor
make start
make status
make verify
正在检查本地 Dashboard...
如果服务还没启动，桌面 App 会停留在这个引导页，直到本地 Dashboard 可访问。
候选地址: http://127.0.0.1:8080-8089
