# Security Policy / 安全说明

## English

If you discover a security issue, do not open a public issue with live secrets,
tokens, webhook URLs, or private machine details.

Please report the issue privately first and include:

- affected file or endpoint
- impact
- reproduction steps
- whether secrets may already be exposed in git history

Before publishing this repository, rotate any previously exposed webhook or token.

## 中文

如果你发现安全问题，不要在公开 issue 中直接贴出真实 secret、token、
webhook URL 或机器私有信息。

请优先私下报告，并提供：

- 受影响的文件或接口
- 影响范围
- 复现步骤
- 是否已经进入 git 历史

在公开仓库前，请先轮换所有曾经暴露过的 webhook 或 token。
